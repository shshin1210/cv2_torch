# Opencv functions to Pytorch

We have the following opencv's functions implemented in pytorch (Thanks for the help from : OpenAI. (2026). ChatGPT (5.3) [Large language model]. https://chat.openai.com/chat)

- undistortPoints
- undistortPoints_normalized
- convertPointsToHomogeneous
- Rodrigues
- projectPoints
- remap
- reprojectImageto3D
- initUndistortRectifyMap
- stereoRectify

You may just import cv2_torch to use such functions!

We have simple tests to compare the results between cv2_torch and opencv
